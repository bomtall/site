About

Tom is a second year university student and Data Analyst working for Skanska UK and studying at London South Bank University.

For his Web Technology module he has created this website. The purpose of this website is for others to learn about graph transformations using an interactive graph and complete a quiz. Graph transformations can be tricky to learn and remember, by using this tool, it may make the process quicker and easier.

The website also hosts Tom's CV and some examples of his work. The website has been created using HTML, CSS, JavaScript and utilises the JavaScript libraries D3.js and jQuery. The code for the website is open-source and hosted on GitHub. You can check out the repository for yourself here.

Tom is originally from Yorkshire but is living in London and is a keen rock climber check out his CV for more information or reach out and contact using the links in the footer of this page.
