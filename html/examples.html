Animating Telemetry Data Using Python

Blog Entry 01/12/2022 - Animating Data using MatPlotLib

One of the workshops I attended at EuroPython 2022 in Dublin was all about animating data using MatplotLib and Python. The workshop was put on by Stephanie Molin, author of the book Hands-On Data Analysis with Pandas. I really enjoyed the workshop which showed me lots of new things including the value in knowing libraries in-depth. The deep dive she gave into MatPlotLib was really insightful and thorough.

Obviously, I was keen to use my new skills when I got back to work. The first opportunity that came up, was some telemetry data which needed summarising. I was emailed half a million data points in an Excel document. The data was temperature readings, and the team were aware that the temperature had been exceeding a limit that it should not be. They wanted to visualise and summarise this data. I created a Power BI report with some graphs and slicers as well as some summary statistics calculated using Python. I wanted to take it a step further though using my new knowledge learned at EuroPython. I decided to animate the data using MatPlotLib and create a .mp4 file which I then hosted on Azure blob storage. It was the first time I hosted anything on Azure myself and this was cool too, since I can generate a URL and SAS token, embed the video in an iframe and include it on this website.
