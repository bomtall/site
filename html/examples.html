Animating Telemetry Data Using Python

Blog Entry 01/12/2022 - Animating Data using MatPlotLib

One of the workshops I attended at EuroPython 2022 in Dublin was all about animating data using MatplotLib and Python. The workshop was put on by Stephanie Molin, author of the book Hands-On Data Analysis with Pandas. I really enjoyed the workshop which showed me lots of new things including the value in knowing libraries in-depth. The deep dive she gave into MatPlotLib was really insightful and thorough.

Obviously, I was keen to use my new skills when I got back to work. The first opportunity that came up, was some telemetry data which needed summarising. I was emailed half a million data points in an Excel document. The data was temperature readings, and the team were aware that the temperature had been exceeding a limit that it should not be. They wanted to visualise and summarise this data, focusing on quantifying the amount of time spend above the threshold. I created a Power BI report with some graphs and slicers as well as some summary statistics calculated using Python. I wanted to take it a step further though using my new knowledge learned at EuroPython. I decided to animate the data using MatPlotLib and create a .mp4 file which I then hosted on Azure blob storage. It was the first time I hosted anything on Azure myself and this was cool too, since I can generate a URL and SAS token, embed the video in an iframe and include it on this website.

Blog Entry 20/12/2022 - Learning JavaScript

I learned to code using Python and have enjoyed making Python GUIs, programs, handling data and visualising data using Python. However, I have always wanted to have complete control of data visualisation in the browser, I also wanted to experiment with the shareability that JavaScript offers due to its ability to execute using just a browser.

That's why when I started the web technology module on my university course, I was excited to finally get going with JavaScript. I bought a book, which is free to read digitally here. I haven't finished the book yet, but I highly recommend it because it got me started very quickly and gave me the foundation to build this website. As soon as I had learned enough to feel like I could take on a course in D3.js I went for it, because visualising data was one of my key goals for this coursework. I get access to LinkedIn Learning for free with my academic email address, it is a useful resource with a lot of good content.
