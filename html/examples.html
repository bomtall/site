PowerApps Development - Trade Contractor League Table

One of the tools available to me at Skanska is PowerApps. I can use this low-code app building software to create internal applications which improve data collection and validation within the business. Writing code to validate inputs within the app and storing the results in SQL Databases means that I have trusted data for analysis and automated report building. Creating interactive reports in Power BI, leadership have actionable information and trends to monitor performance. Project teams can discuss KPI scores with contractors and work to make improvements together. Posters created from the Power BI reports are displayed on sites for contractors to understand where they stack up and how their performance is percieved by Skanska. Success of the project means that Directors from the contracting companies have requested the reports to monitor performance.
